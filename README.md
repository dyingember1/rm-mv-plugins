# rm-mv-plugins
Some useful plugins for RPG Maker MV.

## intro
I like RPG Maker games. Wanted to create my own games. Needed some plugins. I did find some 
good ones. But, finding what I needed and all the licensing stuff was a hassle.
So, I decided to create some myself.

Read full lincense on lisence tab.

## How to use?
1. Download what you need.
2. Copy and paste the files in /js/plugins inside your project/game directory.
3. Import and enable them from the Plugin Manager of RPG Maker.
4. Customise and use however you like.
